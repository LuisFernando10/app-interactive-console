Document Etiqueta estandarizada que indica a los navegadores que se trabajará con HTML5 Es la etiqueta principal en la que se incluirá todo el contenido de una página web. Es la etiqueta en la cual se incluirán los metadatos (Los metadatos son aquellas configuraciones que no son visibles en el cuerpo de la página pero que son instrucciones para los navegadores), en él va por ejemplo: - Especificaciones para que la página se visualice en dispositos móviles - Tipo de acentuación con la que trabajarán los textos de la página - Especificaciones para determinados navegadores - Título de la página - Inclución de archivos externos y librerías Es la etiqueta en la que irá todo el contenido visible en la página web, en ellos podemos encontrar: - Párrafos

=> Dentro de esta etiqueta irá el texto de una página web - Salto de línea
=> Esta etiqueta se encarga de insertar un espacio entre párrafos o textos - Línea horizontal
=> Agrewga una línea horizontal a un contenido - Texto preformateado 
 => Esta etiqueta formatea un determinado texto que resalta y se diferencia de los demás

    Los atributos o propiedades, son especificaciones que modifican características de las etiquetas y permiten un funcionamiento en concreto
     -  => 
       src: Es la ruta hacia la imagen que se desea mostrar, ejemplo: imágenes/imagen.png 
       alt: Es una descripción de la imágen en el caso que ésta no cargue en el navegador.
       width: Modifica el ancho de una imágen, generalmente ésta medida está dada en pixeles.
       heigth: Modifica el alto de una imágen , generalmente ésta medida está dada en pixeles.
       class: Este atributo recibe cadenas de texto que me permitirán interactuar con estilos CSS o funcionalidades Java Script
